---
name: plannotator-review
description: Review current uncommitted changes in a visual UI with annotation support
---

Use the plannotator_review tool to open a visual code review UI for the current uncommitted changes.

The user will be able to:
- View the diff of all uncommitted changes
- Add annotations and comments to specific lines
- Approve or request changes

After the review is complete, address any feedback provided by the user.
