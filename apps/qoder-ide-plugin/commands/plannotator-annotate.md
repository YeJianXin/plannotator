---
name: plannotator-annotate
description: Open a markdown file in the annotation UI for review
---

Use the plannotator_annotate tool to open the specified markdown file in a visual annotation UI.

Usage: /plannotator-annotate <file_path>

The user will be able to:
- View the markdown rendered
- Add annotations, comments, and suggestions
- Provide structured feedback

After the annotation session is complete, address any feedback provided by the user.
